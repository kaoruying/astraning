R1｜取得名單
主任要做什麼
建立緣故名單、推薦人名單，讓增員不是想到才做，而是有來源、有紀錄、有追蹤。

常見卡點

課程補強
緣故名單盤點、推薦人名單建立、增員名單分類。

一句提醒
名單不是等出來的，是整理出來、問出來、累積出來的。

R2｜啟動接觸
主任要做什麼
透過自然話題、行為模式測驗或活動邀約，開啟與準增的第一步接觸。

常見卡點
怕太像推銷、不知道怎麼開口，或接觸後不知道如何延續。

課程補強
定聯分級、行為模式測驗應用、準增接觸話術。

一句提醒
好的接觸，不是急著說服，而是先讓對方願意聊下去。

R3｜發掘需求
主任要做什麼
從對方的工作、收入、時間、成長期待中，找出真正重視的工作條件。

常見卡點
聊很多，但沒有問到關鍵；知道對方現況，卻沒有挖出轉換動機。

課程補強
六大面向提問、需求引導、工作價值釐清。

一句提醒
準增不是被說服的，而是被問到自己也開始思考。

R4｜邀約推進
主任要做什麼
根據準增需求，用合適的理由邀約活動、面談或下一步接觸。

常見卡點
邀約常被「我再想想」「沒時間」「先不用」擋住。

課程補強
FAB 邀約法、活動邀約、下一步推進設計。

一句提醒
邀約不是把人拉來，而是讓對方看見下一步值得了解。

R5｜面談追蹤
主任要做什麼
在面談後持續追蹤，釐清對方疑慮，協助準增跨過猶豫點。

常見卡點
對方說再看看就停住，不知道該怎麼追、不敢追、追了又怕有壓力。

課程補強
異議問題處理、追蹤節奏、活動量管理。

一句提醒
追蹤不是催促，而是陪對方把問題想清楚。

R6｜新人報聘
主任要做什麼
協助準增完成報聘前的確認，並銜接新人起步與後續陪跑。

常見卡點
快成交卻鬆掉，或新人進來後缺少明確起步安排。

課程補強
報聘推進、關鍵行動確認、新人起步節奏。

一句提醒
報聘不是結束，而是主任開始帶人的第一步。
